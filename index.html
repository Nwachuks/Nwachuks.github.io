Datify

Datify is a dating app that helps people find love matches. Users update their profile with their personal info. They can also upload their photos, set a main photo and change it as desired. Users can like, send and receive messages to/from their matches to indicate their interest. They can also see when their messages have been read. This was built with Angular and DotNet Core.

Features
  • Server Side Caching
  • Authentication
  • Private Messaging System
  • Photo Uploads
  • Paging, Sorting and Filtering
  • Manage User/Admin Roles

Buyify

Buyify helps users effectively manage their e-commerce activities. Users can view products and product details, and add products to cart. Admins can perform CRUD functionalities on products, stores, brands and merchants. Payments are sorted using an integrated Paystack online payment solution. This was built with the MEAN stack (MongoDB, Express, Angular and NodeJS).

Features
  • Server Side Caching
  • Role based authentication
  • Manage User/Admin Roles
  • Photo Uploads
  • Sort Products
Visit Buyify Github
